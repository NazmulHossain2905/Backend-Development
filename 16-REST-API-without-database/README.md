# **16 - Lesson: REST API Without Database**

### **Routes**

**Base Url**

```javascript
http://localhost:4000
```

### **All Methods**

**GET** - Home route

- http://localhost:4000/

**GET** - Get all users

- /api/users

**POST** - Create a new user

- /api/users

**PATCH** - Update a user

- /api/users/:id

**DELETE** - Delete a user

- /api/users/:id

### _CRUD APPLICATION_

## Run Locally

Clone the project

```bash
  git clone https://github.com/NazmulHossain2905/Backend-Development.git
```

Go to the project directory

```bash
  cd 16-REST-API-without-database
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```
