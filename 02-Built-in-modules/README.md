**Some built-in modules**

### **`fs`** module method

```javascript
fs.writeFile(); // create a new file
fs.readFile();
fs.appendFile(); // update file
fs.rename();
fs.unlink(); // delete file
fs.deleteFile();
fs.exists();
```
